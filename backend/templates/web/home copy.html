Mapping World Ecosystems

Our website aims to develop a method to map world ecosystems as interaction networks, using field data, numeric modeling and collaborative work. Think of this site as an online Delphi panel for ecological network construction, or an attempt to apply crowd sourcing to build (ecological) networks. This is a beta version.
